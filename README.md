# PARADAS DE GUAGUA DE LAS PALMAS

Para esta práctica he decido, a partir de la enseñada en clase sobre las sitycletas, hacer un mapa de las palmas donde se muestren las diferentes paradas de guagua.

He utilizado un mapa nuevo extraido desde OpenStreetMap y las ubicaciones de las guaguas las he obtenido mediante la pagina de [Overpass turbo](https://overpass-turbo.eu/) que, mediante un codigo que me ayudó a obtener ChatGPT, te otorga los datos de las ubicaciones de las diferentes paradas.

Además, si se hace click en uno de los puntos del mapa, estos te muestran algo de información sobre dicha parada.

Por último, cada parada tiene un color, las azules son las paradas de Global y las amarillas las pertenecientes a las guaguas municipales.
