# PARADAS DE GUAGUA DE LAS PALMAS

Para esta práctica he decido, a partir de la enseñada en clase sobre las sitycletas, hacer un mapa de las palmas donde se muestren las diferentes paradas de guagua.

He utilizado un mapa nuevo extraido desde OpenStreetMap y las ubicaciones de las guaguas las he obtenido mediante la pagina de [Overpass turbo](https://overpass-turbo.eu/)
